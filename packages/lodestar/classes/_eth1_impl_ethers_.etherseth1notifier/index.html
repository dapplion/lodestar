    Class: EthersEth1Notifier <T, U, V>¶

    The EthersEth1Notifier watches the eth1 chain using ethers.js

    It proceses eth1 blocks, starting from block number depositContract.deployedAt, maintaining a follow distance. It stores deposit events and eth1 data in a IBeaconDb resumes processing from the last stored eth1 data
